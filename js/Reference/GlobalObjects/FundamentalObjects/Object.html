# Object

对象函数

# 构造器

// 对象初始化器（Object initialiser）或对象字面量（literal）
{ [ nameValuePair1[, nameValuePair2[, ...nameValuePairN] ] ] }

// 以构造函数形式来调用
new Object([value])
参数 描述
nameValuePair1, nameValuePair2, ... nameValuePairN 成对的名称（字符串）与值（任何值），其中名称通过冒号与值分隔
value 任何值

# 静态属性

属性 描述
Object.length 值为 1
Object.prototype 可以为所有 Object 类型的对象添加属性

# 静态方法

# 创建方法

# assign()

通过复制一个或多个对象来创建一个新的对象

Object.assign(target, ...sources)
参数 描述
target 目标对象
sources 源对象
返回值 目标对象

# create()

使用指定的原型对象和属性创建一个新对象

Object.create(proto，[propertiesObject])
参数 描述
proto 新创建对象的原型对象
propertiesObject(可选) 传入一个对象，对象信息参考 属性信息
返回值 一个新对象，带着指定的原型对象和属性

# 操作属性

# 属性信息

参数 描述 默认值
configurable 能否使用delete、能否需改属性特性、或能否修改访问器属性 true
enumerable 属性是否可以被枚举（使用for...in或Object.keys()） true
writable 属性的值是否可以被重写 true
value 属性对应的值,可以使任意类型的值 undefined
get 属性的 getter 函数访问器 undefined
set 属性的 setter 函数访问器 undefined

# 属性配置

# defineProperty

直接在一个对象上定义一个新属性，或者修改一个对象的现有属性，并返回此对象。

Object.defineProperty(obj, prop, descriptor)
参数 描述
obj 定义属性的对象
prop 定义或修改的属性的名称或 Symbol
descriptor 定义或修改的属性描述符
返回值 被传递给函数的对象
# defineProperties()

直接在一个对象上定义新的属性或修改现有属性，并返回该对象。

Object.defineProperties(obj, props)
参数 描述
obj 定义属性的对象
descriptor 定义或修改的属性描述符
返回值 被传递给函数的对象

# 属性访问

# getOwnPropertyDescriptor()

返回指定对象上一个自有属性对应的属性描述符

Object.getOwnPropertyDescriptor(obj, prop)
参数 描述
obj 定义属性的对象
prop 定义或修改的属性的名称或 Symbol
返回值 如果指定的属性存在于对象上，则返回其属性描述符对象，否则返回 undefined。
# getOwnPropertyNames()

返回一个数组，它包含了指定对象所有的可枚举或不可枚举的属性名。

Object.getOwnPropertyNames(obj)
参数 描述
obj 一个对象，其自身的可枚举和不可枚举属性的名称被返回
返回值 在给定对象上找到的自身属性对应的字符串数组
# getOwnPropertySymbols()

返回一个给定对象自身的所有 Symbol 属性的数组

Object.getOwnPropertySymbols(obj)
参数 描述
obj 要返回 Symbol 属性的对象
返回值 给定对象自身上找到的所有 Symbol 属性的数组

# 属性判断

验证方法 描述
Object.isExtensible(obj) 判断对象是否可扩展
Object.isFrozen(obj) 判断对象是否已经冻结
Object.isSealed(obj) 判断对象是否已经密封

# 原型访问

# getPrototypeOf()

返回指定对象的原型（内部[[Prototype]]属性的值）

Object.getPrototypeOf(object)
参数 描述
obj 定义属性的对象
返回值 返回对象的原型。如果没有继承属性，则返回 null 。

# setPrototypeOf()

设置一个指定的对象的原型 ( 即, 内部[[Prototype]]属性）到另一个对象或 null

Object.setPrototypeOf(obj, prototype)
参数 描述
obj 设置其原型的对象
prototype 对象的新原型
返回值 指定的对象

# 防止对象方法

静态方法 返回
freeze() 冻结对象：其他代码不能删除或更改任何属性
preventExtensions() 防止对象的任何扩展
seal() 防止其他代码删除对象的属性

# 验证方法

# is()

Object.is(value1, value2);
参数 描述
value1 比较的第一个值
value2 比较的第二个值
返回值 一个 Boolean 类型标示两个参数是否是同一个值

# 迭代器方法

静态方法 返回
entries() 对象元素的键值对
keys() 对象元素的键数组
values() 对象元素的值数组。

# 实例属性

实例属性 返回
Object.prototype.constructor 构造器
Object.prototype._proto_ 指向当对象被实例化的时候，用作原型的对象
Object.prototype._noSuchMethod_ 当未定义的对象成员被调用作方法的时候，允许定义并执行的函数

# 实例方法

# 转换方法

实例方法 返回
toSource() 对象源代码
toString() 对象字符串
toLocaleString([区域语言[,数据类型]]); 指定对象区域语言和类型的字符串
valueOf() 对象的原始值

# 验证方法

# hasOwnProperty()

obj.hasOwnProperty(prop)

返回一个布尔值 ，表示某个对象是否含有指定的属性，而且此属性非原型链继承的

参数 描述
prop 要检测的属性的 String 字符串形式表示的名称，或者 Symbol。
返回值 判断某个对象是否含有指定的属性的布尔值 Boolean

# isPrototypeOf()

测试一个对象是否存在于另一个对象的原型链上

prototypeObj.isPrototypeOf(object)
参数 描述
object 在该对象的原型链上搜寻
返回值 Boolean，表示调用对象是否在另一个对象的原型链上

# propertyIsEnumerable()

obj.propertyIsEnumerable(prop)

返回一个布尔值，表示指定的属性是否可枚举

参数 描述
prop 需要测试的属性名
返回值 指定的属性名是否可枚举的布尔值

# 关联函数

Object.prototype._defineSetter_()

关联一个函数到一个属性。设置该函数时，执行该修改属性的函数。

Object.prototype._lookupGetter_()

返回使用 _defineGetter_ 定义的方法函数 。

Object.prototype._lookupSetter_()

返回使用 _defineSetter_ 定义的方法函数。
