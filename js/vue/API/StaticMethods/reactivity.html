# 响应性API

# 响应性基础 API

# 响应式

静态方法 描述
reactive 响应式对象
readonly 只读响应式对象
toRaw 返回 reactive 或 readonly 代理的原始对象

# 验证

静态方法 描述
isProxy 检测响应式对象是reactive还是readonly
isReactive 检查对象是否是 reactive创建的响应式 proxy。
isReadonly 检查对象是否是由readonly创建的只读代理
toRaw 返回 reactive 或 readonly 代理的原始对象

# 返回

静态方法 描述
toRaw 返回 reactive 或 readonly 代理的原始对象
markRaw 标记一个对象，使其永远不会转换为代理。返回对象本身

# 原型响应式

静态方法 描述
shallowReactive 响应式原型对象,嵌套深层对象不响应式
shallowReadonly 只读响应式原型对象,嵌套深层对象不响应式

# Refs

静态方法 描述
ref 响应式值
toRef 响应式原型对象
toRefs 将响应式对象转换为普通对象，其中结果对象的每个 property 都是指向原始对象相应 property 的ref
unref 如果参数为 ref，则返回内部值，否则返回参数本身
isRef 验证ref
customRef 工厂函数ref返回一个带有 get 和 set 的对象
shallowRef 创建一个 ref，它跟踪自己的 .value 更改，但不会使其值成为响应式的
triggerRef 手动执行与 shallowRef](#shallowref) 关联的任何效果

# ref

接受一个内部值并返回一个响应式且可变的 ref 对象。

const count = Vue.ref(0)
console.log(count.value) // 0

count.value++
console.log(count.value) // 1

# toRef

源响应式对象上的属性创建一个 ref。然后可以将 ref 传递出去，从而保持对其源 属性的响应式连接

const state = reactive({
  bar: 2
})

const foo = toRef(state, 'foo')

foo .value=2
console.log(state.foo) // 2

# toRefs

将响应式对象转换为普通对象，其中结果对象的每个属性都是指向原始对象相应属性的ref

const state = Vue.reactive({
  foo: 1,
  bar: 2
})

const stateAsRefs = Vue.toRefs(state)
const { foo, bar } = Vue.toRefs(state)

// ref 和 原始property “链接”
state.foo++
console.log(foo)
console.log(stateAsRefs.foo.value) // 2

stateAsRefs.foo.value++
console.log(state.foo) // 
console.log(foo)

# Computed 与 watch

静态方法 描述
computed 自动计算属性
watchEffect 响应式立即运行函数，依赖项更改就会重新运行
watch 侦听器

# computed

使用 getter 函数，并为从 getter 返回的值返回一个不变的响应式 ref 对象

const count = Vue.ref(1)
const plusOne = Vue.computed(() => count.value + 1)

console.log(plusOne.value) // 2

plusOne.value++ // error

get 和 set 函数的对象来创建可写的 ref 对象

const count = Vue.ref(1)
const plusOne = Vue.computed({
  get: () => count.value + 1,
  set: val => {
    count.value = val - 1
  }
})

plusOne.value = 1
console.log(count.value) // 0

# watchEffect

  • 立即执行函数，没有惰性
  • 不需要传递指定侦听的内容，自动感知代码有依赖，不需要传递很多多餐，只需要传递一个回调函数
  • 不能获取之前数据的值
  • 依赖项更改数据变化，就会重新运行
const count = Vue.ref(0)

Vue.watchEffect(() => console.log(count.value))
// -> logs 0

setTimeout(() => {
  count.value++
  // -> logs 1
}, 100)

# watch

  • 具备一定的惰性

  • 参数可以拿到原始值和当前值

  • 侦听多个数据变化，用一个侦听器可以承载

# 侦听一个单一源

侦听器 data 源可以是返回值的 getter 函数，也可以是 ref：

// 侦听一个getter
const state = Vue.reactive({ count: 0 })
Vue.watch(
  () => state.count,
    //当前值和原始值
  (count, prevCount) => {
    /* ... */
  }
)

// 直接侦听一个ref
const count = Vue.ref(0)
Vue.watch(count, (count, prevCount) => {
  /* ... */
})

# 侦听多个源

侦听器还可以使用数组同时侦听多个源：

Vue.watch([fooRef, barRef], ([foo, bar], [prevFoo, prevBar]) => {
  /* ... */
})
