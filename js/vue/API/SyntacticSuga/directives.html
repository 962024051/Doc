# 指令

# 条件指令

指令语法 类型 描述
v-show any 根据表达式的真假值，切换元素的 display样式属性
v-if any 根据表达式的真假值来有条件地渲染元素
v-else 不需要表达式
v-else-if any 充当 v-if 的“else-if 块”，可以连续使用
<div v-show="表达式">内容</div>

<div v-if="表达式">
  A
</div>
<div v-else-if="表达式">
  B
</div>
<div v-else-if="表达式">
  C
</div>
<div v-else>
  Not A/B/C
</div>

# 列表指令

指令语法 类型 描述
v-for Array | Object | number | string | Iterable 基于源数据多次渲染元素或模板块
  • <ul>
  <li v-for="item in items">
    {{ item.value }}
  </li>
</ul>

  • <div v-for="item in items" :key="item.id">
  {{ item.text }}
</div>

  • <ul>
  <li v-for="(item, index) in items">
    {{ item.value }}
  </li>
</ul>

  • <ul>
  <li v-for="(value, key) in object">
    {{ value }}
  </li>
</ul>

  • <ul>
  <li v-for="(value, name, index) in object">
    {{ value }}
  </li>
</ul>

# 事件指令

指令语法 类型 缩写 描述
v-on Function | Inline Statement | Object @ 绑定事件监听器。
<!-- 方法处理器 -->
<button v-on:事件名="方法或属性"></button>

<!-- 动态事件 -->
<button v-on:[事件名]="方法或属性"></button>

<!-- 内联语句 -->
<button v-on:事件名="函数表达式"></button>

<!-- 多事件处理器 -->
<button @click="函数表达式1, 函数表达式2"></button>

<!-- 缩写 -->
<button @事件名="方法或属性"></button>

<!-- 动态事件缩写 -->
<button @[事件名]="方法或属性"></button>

# 修饰符

<!-- 修饰符 -->
<button @事件名.修饰符="表达式"></button>

  • 事件修饰符

    <!-- 事件将只会触发一次 -->
<a @事件名.once="表达式"></a>

<!-- 停止冒泡 -->
<button @事件名.stop="表达式"></button>

<!-- 阻止默认行为 -->
<form @事件名.prevent="表达式"></form>

<!-- 修饰符可以串联 -->
<a @事件名.stop.prevent="表达式"></a>

<!-- 只有修饰符 阻止默认行为，没有表达式-->
<form @事件名.prevent></form>

<!-- 添加事件监听器时使用事件捕获模式 -->
<!-- 即内部元素触发的事件先在此处理，然后才交由内部元素进行处理 -->
<div @事件名.capture="表达式">...</div>

<!-- 只当在 event.target 是当前元素自身时触发处理函数 -->
<!-- 即事件不是从内部元素触发的 -->
<div @事件名.self="表达式">...</div>

    <!-- 滚动事件的默认行为 (即滚动行为) 将会立即触发   -->
<!-- 而不会等待 `onScroll` 完成                   -->
<!-- 这其中包含 `event.preventDefault()` 的情况   -->
<div @scroll.passive="onScroll">...</div>

  • 按键修饰符

    在监听键盘事件时，我们经常需要检查详细的按键

    <input @事件名.按键修饰符="表达式" />
    按键别名 描述
    .enter 输入回车
    .tab 制表符
    .delete “删除”和“退格”
    .esc 退出
    .space 空格
    .up 上
    .down 下
    .left 左
    .right 右

  • 系统修饰键

    • 键盘修饰键

      按键别名 描述
      .ctrl 控制
      .alt 交替换档键
      .shift 上档转换键
      .meta windows键

    • .exact 修饰符

      允许你控制由精确的系统修饰符组合触发的事件

      <!-- 即使 Alt 或 Shift 被一同按下时也会触发 -->
<button @事件.ctrl="表达式">A</button>

<!-- 有且只有 Ctrl 被按下的时候才触发 -->
<button @事件.ctrl.exact="表达式">A</button>

<!-- 没有任何系统修饰符被按下的时候才触发 -->
<button @事件.exact="onClick">A</button>

    • 鼠标按钮修饰符

      按键别名 描述
      .left 鼠标左键
      .right 鼠标右键
      .middle 鼠标中间键

# 自定义事件

  • 传参

    app.component(自定义事件名称, {
  props: {
    参数名: String
  },
  template: `
    <input 
      type="text"
      :value="参数名"
      @input="$emit('update:参数名', $event.target.value)">
  `
})

<自定义事件名称 v-model:参数名="参数值"></自定义事件名称>

  • 多个 v-model 绑定

    app.component('自定义事件名称', {
  props: {
    参数1: String,
    参数2: String
  },
  template: `
    <input 
      type="text"
      :value="参数1"
      @input="$emit('update:参数1', $event.target.value)">

    <input
      type="text"
      :value="参数2"
      @input="$emit('update:参数2', $event.target.value)">
  `
})

<自定义事件名称
  v-model:参数1="参数值1"
  v-model:参数2="参数值2"
></自定义事件名称>

  • 自定义修饰符

    • 默认修饰符

      <div id="app">
  <my-自定义 v-model.capitalize="参数名"></my-自定义>
  {{ myText }}
</div>

const app = Vue.createApp({
  data() {
    return {
      参数名: ''
    }
  }
})

app.component('my-自定义', {
  props: {
    modelValue: String,
    modelModifiers: {
      default: () => ({})
    }
  },
  methods: {
    emitValue(e) {
      let value = e.target.value
      //判断是否自定义修饰符
      if (this.modelModifiers.capitalize) {
      	//执行自定义语句
        value = value.charAt(0).toUpperCase() + value.slice(1)
      }
      //绑定事件
      this.$emit('update:modelValue', value)
    }
  },
  template: `<input
    type="text"
    :value="modelValue"
    @input="emitValue">`
})

app.mount('#app')

    • 带参数的 v-model 绑定

      <my-component v-model:参数名.capitalize="bar"></my-component>
 
app.component('my-component', {
  props: ['参数名', '参数名Modifiers'],
  template: `
    <input type="text" 
      :value="参数名"
      @input="$emit('update:参数名', $event.target.value)">
  `,
  created() {
    console.log(this.参数名oModifiers) // { capitalize: true }
  }
})
