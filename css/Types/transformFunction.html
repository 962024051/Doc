# 矩阵

# 2D 矩阵

由指定的 6 个值组成的 2D 变换矩阵。这种矩阵的常量值是隐含的，而不是由参数传递的；其他的参数是以列优先的顺序描述的

matrix(a, b, c, d, tx, ty)
matrix3d(a, b, 0, 0, c, d, 0, 0, 0, 0, 1, 0, tx, ty, 0, 1)
matrix( scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY() )
属性 描述
a b c d <number>数据类型描述线性变换
tx ty <number>数据类型描述变换的量

# 3D 矩阵

4x4 齐次矩阵的形式定义一个 3D 转换

matrix3d(a1, b1, c1, d1, a2, b2, c2, d2, a3, b3, c3, d3, a4, b4, c4, d4)
属性 描述
a1 b1 c1 d1 a2 b2 c2 d2 a3 b3 c3 d3 d4 <number>数据类型描述线性变换
a4 b4 c4 <number>数据类型描述变换的量

# 透视

到 z=0 的坐标轴，使用者能够得到 3D 定位元素的透视，每个 3D 元素在 Z 轴的位置大于零意味着离使用者更近，因此会变得更大，同理当元素 Z 周小于零时会变得

perspective(d)
属性 描述
d <length>数据类型距离 z 坐标的距离，如果它是 0 或负值，则不应用透视变换

# 旋转

# 2D 旋转

rotate(a)
rotate(<angle>)
属性 描述
a <angle>旋转的角度数据类型。 正角度表示了顺时针的旋转，负角度表示逆时针的旋

# 3D 旋转

rotate3d(x, y, z, a)
属性 描述
x <number> 类型，可以是 0 到 1 之间的数值，表示旋转轴 X 坐标方向的矢量
y <number> 类型，可以是 0 到 1 之间的数值，表示旋转轴 y 坐标方向的矢量
z <number> 类型，可以是 0 到 1 之间的数值，表示旋转轴 z 坐标方向的矢量
a <angle> 类型，表示旋转角度。正的角度值表示顺时针旋转，负值表示逆时针旋转

# Y 轴旋转

rotateX(a)
属性 描述
a <angle>旋转的角度数据类型。 正角度表示了顺时针的旋转，负角度表示逆时针的旋

# Y 轴旋转

rotateY(a)
属性 描述
a <angle>旋转的角度数据类型。 正角度表示了顺时针的旋转，负角度表示逆时针的旋

# Z 轴旋转

rotateZ(a)
属性 描述
a <angle>旋转的角度数据类型。 正角度表示了顺时针的旋转，负角度表示逆时针的旋

# 缩放

# 2D 缩放

scale(sx)
或
scale(sx, sy)
属性 描述
sx <number>数据类型。 缩放向量的横坐标
sy <number>数据类型。 缩放向量的纵坐标。
如果未设置，则他的默认值被设置为 sx。 从而使得元素在保持原有形状下均等缩放

# 3D 缩放

scale3d(sx, sy, sz)
属性 描述
sx <number>数据类型。 缩放向量的横坐标
sy <number>数据类型。 缩放向量的纵坐标
sz <number>数据类型。 缩放向量的 z 轴坐标

# X 轴缩放

scaleX(s)
属性 描述
s <number>数据类型。 缩放向量的横坐标

# Y 轴缩放

scaleY(s)
属性 描述
s <number>数据类型。 缩放向量的纵坐标

# Z 轴缩放

scaleZ(s)
属性 描述
s <number>数据类型。 缩放向量的 z 轴坐标

# 倾斜

# 2D 倾斜

skew(ax)

skew(ax, ay)
属性 描述
ax <angle>数据类型。 沿横坐标扭曲元素的角度
ay <angle>数据类型。 沿纵坐标扭曲元素的角度。如果未定义，则其默认值为 0，导致纯水平倾斜。

# X 轴倾斜

skewX(a)
属性 描述
a <angle>数据类型。 沿横坐标扭曲元素的角度

# Y 轴倾斜

skewY(a)
属性 描述
a <angle>数据类型。 沿纵坐标扭曲元素的角度

# 移动

# 2D 移动

 translate(tx, ty);
属性 描述
tx <length>和<percentage>和数据类型。 沿横坐标移动元素的宽度
ty <length>和<percentage>数据类型。 沿纵坐标移动元素的高度。
如果未定义，则其默认值为 0，水平移动

# 3D 移动

 translate3d(tx, ty,tz);
属性 描述
tx <length>和<percentage>和数据类型。 沿横坐标移动元素的宽度
ty <length>和<percentage>数据类型。 沿纵坐标移动元素的高度
ty <length>数据类型。 沿Z坐标移动元素的深度

# X轴移动

 translateX(t);
属性 描述
t <length>和<percentage>和数据类型。 沿横坐标移动元素的宽度

# Y轴移动

 translateY(t);
属性 描述
t <length>和<percentage>和数据类型。 沿纵坐标移动元素的高度

# Z轴移动

 translateZ(t);
属性 描述
t <length>数据类型。 沿Z坐标移动元素的深度
